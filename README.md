# Student-Enrollment-Form

I constructed a form using HTML, CSS, and JavaScript, and I use a JsonPowerDB to save data programmatically. Through this project, I want to learn API testing in conjunction with JsonPowerDB. The combination of these tools enables me to investigate and comprehend the complexities of communicating with a JSON-based database, gaining significant insights into the realm of API testing. This practical application broadens my knowledge of web development by allowing me to dynamically manage data via a form and connect with a backend database using JavaScript. This hands-on experience allows me to get real understanding of dealing with APIs and databases, which adds to my total skill set in web development and testing.
